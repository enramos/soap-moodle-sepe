soap-moodle-sepe
================

Servicio web para el seguimiento de acciones formativas en Moodle por parte del SEPE mediante SOAP y WSDL.

El objetivo es una aplicación que cumpla con las especificaciones descritas por el SEPE.

## Especificaciones
